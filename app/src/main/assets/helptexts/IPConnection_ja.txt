(IPConnection_ja.txt)

LAN内で、また環境が許せば(Wi-Fiスポット間など)インターネットで、
対戦できます
Android 4 以降では Wi-Fiダイレクトで 近距離直接対戦できます
但し、WiFi と WiFiダイレクトは同時使用はデバイス依存です
混用すると IP接続が非常に遅くなるかも知れません
そのような場合、接続中のセッションを "切断"し
WiFiダイレクトをリセットしてやり直してください
(プレイ-->Wi-Fiダイレクト-->設定からシステムの設定に入り
あるいは機内モードをOn,Offする)

このデバイスの IP アドレス : IP アドレス
  これを指定してパートナーが接続要求を出す
  Wi-Fiダイレクトでペアを設定すると複数個表示される
  N/A と成っている場合は android の 設定で Wi-Fi が ON か
  確認してください
接続中のセッション : パートナーの IP アドレスを表示
ポートを開く : 接続要求を受け付けるサーバーとして入口を開ける
  "ダイレクト"の場合はサーバー側は"ポートを開く"を押してください
  他のアプリと競合しない確認が取れれば
  省略値のポート番号(=6974)は変更できます
ポートを閉じる : "ポートを開く" と接続されるまでの間
  ボタンがこれに変わります
接続 : リストから接続先を選択した後,ボタンを押してください
  "ダイレクト"の場合はクライアント側から"接続"してください
  失敗したら、相手が"ポートを開く"状態か確認してください
  また"ダイレクト"の場合は"WiFiダイレクトダイアログ"で
  "探索","ペア設定"をやり直してみてください
切断 : 接続を落とします
  このボタンは "接続"されたあとに表示されます
対戦 : 対戦設定ダイアログを表示します
  接続が成功すると使用可能になります
パートナーリスト :
  "Direct-" とあるものはWi-Fiダイレクトでペアリングされたものです
  ペアが設定されるまでは使用できません
編集/追加/削除/削除戻し/↑/↓ : 接続先リストを操作します
  編集/追加でダイアログがポップアップします
  接続要求先の IP アドレス、ポート番号を登録してください
