フリー将棋盤は詰将棋、棋譜並べ用です
"開始"の前後でボタンが変わります
"開始"前は画面下の各種ボタンで駒の配置をいます
"開始"のダイアログでタイマーがせっていできます
あとはローカル対戦の要領で先手後手交互に指します
"中断"ボタンが現れるので、戻って駒の再配置が出来ます
"開始"は必要なわけではありませんが途中の図を保存できます

("棋譜再生"から"変化図"で来た場合下端ボタンが変わります)
-----
"開始" 前
  "配置" : 平手戦の初期配置に並べなおします
  "クリア" : "王"以外全て盤から除きます
     他の駒は持ち駒台から配置してください
  "残後手" : 盤上、先手持ち駒台 にない駒全てを後手の持ち駒とします
    これは詰将棋の条件とされることがある
  "回転" : 盤上の選択された駒を180度回転させ相手側の駒にします
  "裏返" : 選択された駒を裏返して成/不成 を変更します
  "削除" : 選択された駒を盤上から削除
     盤上で駒を払う(フリック)操作でも削除できます
  "Ｃ" : 持ち駒数を全てクリアーします
  駒のアイコン : 選択して盤上の開いている升目にその駒を置けます
    連続してタッチすると駒数を +1 します
    -1 するには選択した状態で "削除" ボタンを押します

"開始" 後
  "保存" : 現図をメモリーに保存します
    開始時の図は自動で保存されています
    フリー将棋盤を閉じると消えます
    ファイルに保存するにはファイルメニューから保存してください
  "前図" : １つ前の図に戻します
  "次図" : 前図で戻ったあとは次図で進めます
  "初めから" : 最初の"開始"状態の配置に戻します

-----
"成"/"不成"ボタン : 初期値は"成"です、再度押すまで有効です
  やり直し出来ません、駒を動かす前に設定してください
  "開始"後に現れます
座標ボタン : 右端の空色のボタン。
  ツータッチモードでの２度目のタッチの変わりに使用します
  この位置で決定という意味です
-----
"ファイル" : ファイル メニューを開き、以下の処理をします
   スナップショットの保存/ロード、
   詰将棋ファイルのロード、
   詰将棋をクリップボードから設定,
   フリー将棋盤を閉じる
   ファイルを読むとメモリーに保存したスナップショットは消されます
"後手開始" : 駒の配置が完了したら、押してローカル対戦モードにし
   後手から始めます
   最初の"開始"のとき"フリー将棋盤設定"ダイアログがでます
   制限時間などを設定できます
"先手開始" : 駒の配置が完了したら、押してローカル対戦モードにし
   先手から始めます
   最初の"開始"のとき"フリー将棋盤設定"ダイアログがでます
"中断" : "開始"前に戻り、駒の配置をやり直す時に押します
"設定" : サウンドなど将棋盤の各種オプションの設定
----
"棋譜再生"から"変化図"で来た場合のボタンは
"閉" : "棋譜再生"を終了します
"棋譜" : "棋譜再生"に戻ります
